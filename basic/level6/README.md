## Encryption

To encode strings, something like a **Caesar Cipher** is used, but with a *variable shift*.
That is, for each subsequent character, the shift will be incremented.

Examples:

- `ABC` -> `ACE`
- `000` -> `012` 
- `BCA` -> `BBB`
